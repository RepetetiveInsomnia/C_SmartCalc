The program is implemented with QT GUI. The calculator has the ability to calculate the result of the entered expressions with features listen in calculator's keys. Additionaly you can display graph with x function.

A graph will be built with default values(from 10 to -10) where you you will be able to view the graph with a possibility of scaling.

Install

In order to start using the calculator, it must be installed using the make install command.

Uninstall

Starts uninstallation process.

Dist

To make an archive of the programm use this.
